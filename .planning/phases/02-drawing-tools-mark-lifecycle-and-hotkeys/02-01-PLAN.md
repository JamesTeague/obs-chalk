Modify FreehandMark to implement two Phase 2 improvements:

**1. Linear interpolation in add_point() (TOOL-01):**
When a new point arrives, compute distance from the last stored point using `std::hypot`. If distance exceeds a step threshold (2.0f pixels), insert intermediate points via linear interpolation between the previous point and the new point. This fills gaps from fast mouse movement.

```cpp
void FreehandMark::add_point(float x, float y) {
    if (!points_.empty()) {
        float dx = x - points_.back().x;
        float dy = y - points_.back().y;
        float dist = std::hypot(dx, dy);
        const float STEP = 2.0f;
        if (dist > STEP) {
            float ox = points_.back().x, oy = points_.back().y;
            int n = static_cast<int>(dist / STEP);
            for (int i = 1; i <= n; ++i) {
                float t = static_cast<float>(i) / static_cast<float>(n + 1);
                points_.push_back({ox + dx * t, oy + dy * t});
            }
        }
    }
    points_.push_back({x, y});
}
```
Add `#include <cmath>` to freehand-mark.cpp for `std::hypot`.

**2. distance_to() implementation (for pick-to-delete):**
Replace the `hit_test` override with `distance_to`. Compute minimum point-to-line-segment distance across all consecutive segment pairs in the polyline:
- For each segment from points_[i-1] to points_[i], compute the closest point on the segment to the query (x, y) using the standard projection formula with clamped t parameter
- Track minimum distance across all segments
- Return `std::numeric_limits<float>::max()` if fewer than 2 points

Reference formula per segment (A=points_[i-1], B=points_[i]):
```
dx = B.x - A.x, dy = B.y - A.y
len2 = dx*dx + dy*dy
if len2 < 1e-6: dist = hypot(x - A.x, y - A.y)
else: t = clamp(((x-A.x)*dx + (y-A.y)*dy) / len2, 0, 1)
      dist = hypot(x - (A.x + t*dx), y - (A.y + t*dy))
```

Add `#include <limits>` and `#include <algorithm>` to freehand-mark.cpp.

Update freehand-mark.hpp: replace `bool hit_test(float x, float y) const override` with `float distance_to(float x, float y) const override`.
