2. **src/tool-state.hpp** -- Minimal tool state stub for Phase 1:
   ```cpp

   #include <cstdint>

   enum class ToolType : uint8_t {
       Freehand = 0,
       // Arrow, Circle, Cone, Laser added in Phase 2
   };

   struct ToolState {
       ToolType active_tool = ToolType::Freehand;
       uint32_t active_color = 0xFFFFFFFF; // ABGR white
   };
   ```

3. **src/chalk-source.hpp** -- ChalkSource struct:
   ```cpp
   #pragma once
   #include <obs-module.h>
   #include "tool-state.hpp"

   struct ChalkSource {
       obs_source_t *source;
       ToolState tool_state;

       explicit ChalkSource(obs_source_t *src) : source(src) {}
   };
   ```
   Note: MarkList member added in Plan 02 when the mark types exist.

4. **src/chalk-source.cpp** -- Source implementation with all required callbacks:
   - `chalk_source_info` struct: `.id = "chalk_drawing_source"`, `.type = OBS_SOURCE_TYPE_INPUT`, `.output_flags = OBS_SOURCE_VIDEO | OBS_SOURCE_INTERACTION | OBS_SOURCE_CUSTOM_DRAW`
   - `chalk_get_name()`: returns `obs_module_text("obs-chalk")` (resolves to "Chalk Drawing" from locale)
   - `chalk_create()`: allocates ChalkSource, returns it
   - `chalk_destroy()`: deletes ChalkSource
   - `chalk_get_width()`: uses `obs_get_video_info()` to return `ovi.base_width`
   - `chalk_get_height()`: uses `obs_get_video_info()` to return `ovi.base_height`
   - `chalk_video_render()`: UNUSED_PARAMETER(effect), sets up transparent blend state (gs_blend_state_push, gs_enable_blending, gs_blend_function with SRCALPHA/INVSRCALPHA), gets OBS_EFFECT_SOLID, begins technique, ends technique, pops blend state. No marks to draw yet -- this establishes the render loop structure.
   - `chalk_mouse_click()`: stub (logs or no-op for now)
   - `chalk_mouse_move()`: stub (logs or no-op for now)

**Critical invariants to establish:**
- `OBS_SOURCE_TYPE_INPUT` (not FILTER) -- required for independent overlay positioning and mouse interaction
- `OBS_SOURCE_CUSTOM_DRAW` in output_flags -- required for correct alpha compositing
- `OBS_SOURCE_INTERACTION` in output_flags -- required for mouse events to fire
- `get_width/get_height` return non-zero via `obs_get_video_info` -- without this, source is invisible
- `video_render` NEVER calls `obs_enter_graphics` (already in graphics context)

5. **Update CMakeLists.txt** to include the new source files.

6. **Build the plugin:** `cmake --build build_macos` (or appropriate build dir from preset).
